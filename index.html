Lazy Loading Responsive Images in a CSS Grid Gallery using the <picture> element, Intersection Observer API and Image Placeholders

By ZOODUCK on 02 October 2018

This CSS Grid Gallery Demo illustrates use of the new IntersectionObserver API to load images when they appear in the viewport (and not before).

In this example I have set the threshold property of the IntersectionObserver to 0.5. The observer handler function (assigned to the IntersectionObserver instance) is called when 50% or more of its target element (in this case, the image) is visible in the viewport.
